**Mug color**
---

Retrieves and sets the color of the mug's LED indicator.

* **UUID**

`fc540014-236c-4c94-8fa9-944a3e5353fa`

* **Methods**

`READ`
`WRITE`

* **Data Format**

Size: 4 Bytes
Type: RGBA color code

Byte | Data
--- | ---
0 | Red
1 | Blue
2 | Green
3 | Alpha

* **Example Data**

For setting the mug color to lime-green: `BFFF00FF`
